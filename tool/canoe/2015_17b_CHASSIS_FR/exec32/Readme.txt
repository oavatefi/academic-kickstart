Readme Konverterupdate für CANoe >= 7.6

1. Die DLL VecConv.dll, AutosarFlexRayNM.dll, Can_Fr_Gw.dll und DAGFrIL.dll muss in das Exec32 Verzeichnis der betreffenden CANoe Installation kopiert werden.

2. Löschen der temporären Dateien *.CF und *.FC, die sich im Windows Temp Verzeichnis (%temp%) befinden.
   Um zu diesem Verzeichnis zu gelangen, einfach "%temp%" (ohne Anführungszeichen) in der Adresszeile des Explorers eingeben.
   
   
   
   Readme converter update for CANoe > = 7.6 1. 
The DLL VecConv.dll, AutosarFlexRayNM.dll, Can_Fr_Gw.dll and DAGFrIL.dll must be copied into the directory exec32 the CANoe installation concerned. 
2. Delete the temporary files * and * .CF .FC that in Windows Temp folder ( % temp% ) are located .    
 To go to this directory , just type " % temp% " ( without quotation marks ) in the address bar of the explorer.
